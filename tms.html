Phase 1 Scope of Work

1. Database Architecture

  • Designing and structuring the database to support the application's core.
  • Creation of the essential database tables:
    • Users (Clients, Agents, Drivers, Accounting Team)
    • Invoices (For billing and financial records)
    • Loads (Tracking shipments and assignments)
    • Authentication Logs (User login history for security tracking)
  • Establishing relationships between the tables.

2. Oracle APEX Portal Development

  • User Authentication System:
    • Implementing a secure login mechanism for different user roles.
    • Session management and logout functionality.
  • Client Dashboard (Invoice Viewer):
    • Building the dashboard for clients to log in and view their invoices.
    • Simple, user-friendly interface with filtering options.

3. Security & Performance Enhancements

  • User authentication with encryption for secure access.
  • Data validation and error handling to prevent unauthorized actions.
  • Optimized queries and indexing for system performance.
